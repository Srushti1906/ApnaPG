# 🎯 Enhanced Filter Guide - Browse PGs

Your ApnaPG application now has **advanced filters** to help you find the perfect PG! Here's how to use them:

---

## 🚀 Quick Start: Using Filters

### Option 1: Quick Filter Buttons (Fastest)
Click any of these buttons to instantly apply a pre-set filter:

| Button | What it does |
|--------|--------------|
| 👩 Girls PGs | Shows only girls' PGs |
| 👨 Boys PGs | Shows only boys' PGs |
| 👨‍👩‍👧 Couple/Family | Shows family and couple-friendly PGs |
| 💰 Under ₹5000 | Shows PGs within ₹5000 budget |
| 📶 WiFi Available | Shows PGs with WiFi |
| ❄️ AC Rooms | Shows only PGs with AC |
| 🍽️ With Mess | Shows PGs which have mess available |
| ✅ Verified Only | Shows only verified PGs |

**Tip:** Click a quick filter button to instantly see results. Click "Clear All Filters" to reset.

---

### Option 2: Advanced Filters (Custom Search)

#### 📍 **Location & Basic Info**
- **City**: Search by city name (e.g., "Pune", "Delhi")
- **Area/Locality**: Search by specific area (e.g., "Katraj", "Kothrud")
- **Gender Policy**: Choose who the PG allows
  - Boys Only
  - Girls Only
  - Family/Couple
  - Mixed (any gender)
- **Max Budget**: Enter your maximum budget in ₹ (e.g., 8000)
- **Sort By**: Order results by price
  - Low to High: Cheapest first
  - High to Low: Most expensive first

#### 🏠 **Amenities** (Check what you need)
Select all amenities that matter to you:
- ❄️ **AC** - Air conditioning in rooms
- 📶 **WiFi** - Internet connection
- 🚗 **Parking** - Vehicle parking available
- 🚿 **Attached Bath** - Personal bathroom in room
- 🧺 **Laundry** - Washing machine available
- 📚 **Study Table** - Desk for studying
- 📹 **CCTV** - Security cameras
- ⚡ **Power Backup** - Backup generator/UPS

#### 🛏️ **Stay Options**
- 🍽️ **Mess Available** - Food provided by PG
- 📅 **Daily Stay Available** - Can stay for just a few days (otherwise monthly only)
- ✅ **Verified PG** - PG has been verified by ApnaPG

## 💡 Pro Tips

### 1. **Combine Multiple Filters**
   - Don't just use one filter, combine them!
   - Example: Gender + Budget + Amenities = Perfect PG

### 2. **Start Broad, Then Narrow Down**
   - First search by City and Gender
   - See results
   - Then add Budget and Amenities

### 3. **Use Quick Filters for Common Searches**
   - Quick filters are pre-configured and save time
   - Use them as a starting point

### 4. **Check "Sort By"**
   - Sort by price to compare costs easily
   - Sort Low → High to find cheapest options

### 5. **Filter Count**
   - The counter shows "X PGs found"
   - If it's 0, loosen your filters
   - Remove filters one by one to see what's limiting results

## 🆘 Troubleshooting

### "No PGs Found" - What to do?
1. ✅ Click "Clear All Filters"
2. ✅ Make sure you entered the city name correctly
3. ✅ Try removing budget filter (if set very low)
4. ✅ Uncheck some amenities (might be too restrictive)
5. ✅ Try a different gender policy

### "Too Many Results" - How to narrow down?
1. ✅ Add a budget limit
2. ✅ Select specific amenities you need
3. ✅ Filter by gender policy
4. ✅ Search for specific area/locality
5. ✅ Sort by price

### "Can't find PG in my area"
1. ✅ Check if you spelled the city/area correctly
2. ✅ Try using just the city name without specific area
3. ✅ Look at what areas are available first
4. ✅ Clear filters and browse all PGs
